Programming

Overview:

I currently have four projects on my GitHub: this website, a script I wrote to automate anti-roll bar design, tire analysis for vehicle dynamics, and a script I wrote to generate optimal paths for a given track


Website:

The purpose of this website is to both showcase what I've done and get some practice with front-end web development. This is a learning tool that allows me to work on my web development skills and see progress over time.


Anti-roll bar design:

I initially derived stiffness equations for the ARB geometry I planned to implement. However, I realized I could write an automated calculation system, allowing me to test various designs without having to perform manual design modifications.

I wrote a separate script to filter through acceptable geometry values that would package with the car. I then used this filtration process to find dimensions that maximized stiffness range while preventing yield of components. The code linked is my automated calculation system and a visualization program I wrote to go with it.


Tire Analysis:

In order to make meaningful simulations, you need experimental data to drive model behavior. The tire analysis repository currently processes raw tire data, filters it for meaningful information, and runs a series of fits to generalize the given data points and extrapolate to any set of conditions.


Optimal Path Generation:

My path generation script takes a specified race track (points defining the track and width), and it generates an optimal racing line along that track. This is done by minimizing curvature over the entire length of the path, leading to parameterization of cubic splines, a series of vector calculus operations, multiple iterations of cost functions, and heavy minimization operations.
