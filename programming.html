Anti-roll bar design:

I initially derived stiffness equations for the ARB geometry I planned to implement. However, I realized I could write an automated calculation system, allowing me to test various designs without having to perform manual design modifications.

I wrote a separate script to filter through acceptable geometry values that would package with the car. I then used this filtration process to find dimensions that maximized stiffness range while preventing yield of components. The code linked is my automated calculation system and a visualization program I wrote to go with it.
